# mrs-projekat-tim
Projekat iz predmeta metodologije i razvoj softvera
Sistem za upravljanje sportskim klubovima - Ideja projekta je da omogućimo efikasno vođenje sportskih klubova kroz evidenciju igrača, trenera i utakmica. Sistem podržava: upravljanje igračima, trenerima i utakmicama, evidenciju treninga, povreda i rezultata, statistiku učinka igrača (poeni, asistencije i sl.), kao i upravljanje članarinama i terminima treninga.
